Intuition

INTUITION is a game of games, players must beat 9 different stages of UI-based memory and puzzle games with increasing difficulty to win.

This is my first original game made outside of class. Its made entirely in Geometry Dash and it showcases my skills in programming, optimization, game design, and UI design. In making this game I learned a lot about the process of game development. As for optimization, developing this project also taught me the importance of tackling performance issues and finding causes for the issues that playtesters encountered.

Lacertus

Lacertus is a Rogue-like and Souls-like game made entirely in Geometry Dash. Among many things, this project features:

  • My own combat system that translates the animation based combat of Darksouls into the 2d platform.

  • Seventeen weapons for said combat system each with their own unique normal attack chains, charged attacks, and skills

  • Six different enemies, three flyers and three ground enemies, each one has its own unique form of movement or attacking

  • Three minibosses of increasing difficulty, programmed in way that allows the player to leave the fight if they deem it better to try a different one first

  • Six unique areas with their own platforming challenges and decoration

  • A randomized map created by the program using three of the six areas, random enemy placement, random breakable pot placements and random miniboss placements

  • A combination of frame-by-frame animation and particle effects (as seen in the effects animation), motion animation seen in the minibosses and weapon animations, and frame-by-frame animation used for the final boss

  • A UI system used for the main menu that allows the player to see what weapons and enemies they have unlocked and use said weapons in a training mode. It also has a guide to help players learn the mechanics of the game
All original work, created by myself over the course of eight months, with 1006 hours recorded in the editor.

This project does feature work not created by me: the player icon art, images used for particle effects, music, and sound effects are all assets that were provided by Geometry Dash, in the editor.

EZ Typing [MI 230]

EZ Typing is a short typing game with two twists: you type by clicking keys on the UI keyboard, and you control your cursor like the ship from Asteroids

This game was made for my final project in MI 230: Game Design. The project allowed for complete freedom so I decided to go with an idea I had for a while. My main goal in the game design itself was to make it as annoying as possible while also making the gameplay fun to learn. I didn't know much about scripting in Unity so I learned a lot in that regard through making this project.
