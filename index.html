My Projects

Intuition

INTUTION is a game of games, players must beat 9 different stages of UI-based memory and puzzle games with increasing difficulty to win.

This is my first original game made outside of class. Its made entirely in Geometry Dash and it showcases my skills in programming, optimization, game design, and UI design. However, in a lot of respects this game taught me what not to do, and I have changed how I approach game development because of that. As for optimization, developing this project also taught me the importance of tackling performance issues and figuring out how to approach finding causes for issues that playtesters encountered.

EZ Typing

EZ Typing is a short typing game with two twists: you type by clicking keys on the UI keyboard, and you control your cursor like the ship from Asteroids

This game was made for my final project in MI 230: Game Design. The project allowed for complete freedom so I decided to go with an idea I had for a while. My main goal in the game design itself was to make it as annoying as possible while also making the gameplay fun to learn. I didn't know much about scripting in Unity so I learned a lot in that regard through making this project.
